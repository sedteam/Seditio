# Инструкция по установке и настройке плагина `cookienotice`

Плагин предназначен для информирования посетителей сайта об использовании файлов cookies и сборе статистики с кнопками «Принять все» и «Отказаться».

## Шаг 1. Установка плагина
1. Перейдите в **Панель управления** Seditio.
2. Откройте вкладку **Инструменты / Плагины**.
3. Найдите плагин **Cookie Notice** и нажмите кнопку **Установить**.

## Шаг 2. Настройка плагина
1. После установки перейдите в настройки плагина (вкладка **Конфигурация / Cookie Notice**).
2. Заполните параметры конфигурации:
   - **Текст уведомления**: Вы можете оставить это поле пустым, чтобы использовать стандартный текст из языкового файла, или написать свой собственный. Поле поддерживает плейсхолдеры `{STAT_URL}` и `{POLICY_URL}`, которые будут автоматически заменены на соответствующие ссылки.
   - **Ссылка на страницу сбора статистики**: Укажите URL-адрес информационной страницы (например, `/sborstat`).
   - **Ссылка на Политику конфиденциальности**: Укажите URL-адрес страницы политики конфиденциальности (например, `/policy`).

## Шаг 3. Добавление в шаблон
Для вывода плагина необходимо добавить тег `{FOOTER_COOKIENOTICE}` в шаблон футера.

1. Откройте файл шаблона футера вашего активного скина:
   `skins/<имя_вашего_скина>/footer.tpl`
2. Добавьте тег `{FOOTER_COOKIENOTICE}` перед закрывающим тегом `</body>`. Рекомендуется размещать его перед подключением основных JS-скриптов. Пример:
   ```html
   {FOOTER_COOKIENOTICE}
   {FOOTER_JAVASCRIPT}
   ```
3. Сохраните изменения и очистите кэш сайта в панели управления администратора.
