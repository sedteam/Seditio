# Инструкция по установке и настройке плагина i18n для Seditio CMS

Плагин `i18n` предназначен для организации мультиязычного контента страниц и категорий (структуры). Он позволяет переводить заголовки, описания, тело страниц (включая второе текстовое поле `text2`) и все SEO-метаданные.

---

## 1. Установка и настройка

1. Скопируйте файлы плагина в директорию `plugins/i18n/`.
2. Зайдите в Панель Администратора -> **Плагины** (по адресу `/admin/plug` или `index.php?module=admin&m=plug` без ЧПУ) и установите плагин **i18n**. При установке будут автоматически созданы необходимые таблицы в БД (`{prefix}i18n_pages` и `{prefix}i18n_structure`).
3. В настройках плагина (по адресу `/admin/plug` или `index.php?module=admin&m=plug` без ЧПУ в разделе конфигурации плагина i18n) укажите коды языков перевода через запятую.
   - *Пример:* `en,de`
   - *Примечание:* Язык сайта по умолчанию (например, `ru`) указывать **не нужно**. Он считается базовым.

---

## 2. Интеграция в шаблоны (TPL) темы оформления

Для того чтобы в формах добавления/редактирования страниц и категорий появились вкладки для других языков, необходимо добавить соответствующие теги в ваши `.tpl` файлы.

### А. Добавление страниц (`skins/ваша_тема/page.add.tpl` или `modules/page/tpl/page.add.tpl`)

Найдите блок вкладок (класс `tabs` внутри `sedtabs`) и добавьте тег заголовков вкладок:
```html
<ul class="tabs">
    <li><a href="{PHP.sys.request_uri}#tab1" class="selected">{PHP.L.Page}</a></li>
    <li><a href="{PHP.sys.request_uri}#tab2">{PHP.L.Meta}</a></li>
    <li><a href="{PHP.sys.request_uri}#tab3">{PHP.L.Options}</a></li>
    <!-- Вкладки для перевода -->
    {PAGEADD_I18N_TABS_HEADERS}
</ul>
```

Затем найдите закрывающий тег контейнера вкладок (`tab-box`) и перед ним добавьте тег содержимого вкладок:
```html
<div class="tab-box">
    ... (стандартные вкладки tab1, tab2, tab3) ...

    <!-- Содержимое вкладок перевода -->
    {PAGEADD_I18N_TABS_BODY}
</div>
```

---

### Б. Редактирование страниц (`skins/ваша_тема/page.edit.tpl` or `modules/page/tpl/page.edit.tpl`)

Сделайте аналогичные правки в форме редактирования.

Добавьте заголовки:
```html
<ul class="tabs">
    <li><a href="{PHP.sys.request_uri}#tab1" class="selected">{PHP.L.Page}</a></li>
    <li><a href="{PHP.sys.request_uri}#tab2">{PHP.L.Meta}</a></li>
    <li><a href="{PHP.sys.request_uri}#tab3">{PHP.L.Options}</a></li>
    <!-- Вкладки для перевода -->
    {PAGEEDIT_I18N_TABS_HEADERS}
</ul>
```

Добавьте тело вкладок:
```html
<div class="tab-box">
    ... (стандартные вкладки tab1, tab2, tab3) ...

    <!-- Содержимое вкладок перевода -->
    {PAGEEDIT_I18N_TABS_BODY}
</div>
```

---

### В. Редактирование категорий (`skins/ваша_тема/admin.page.tpl` или `modules/page/tpl/admin.page.edit.tpl`)

Внедрите теги в секцию редактирования настроек категории (`STRUCTURE_UPDATE`).

Добавьте заголовки:
```html
<ul class="tabs">
    <li><a href="{PHP.sys.request_uri}#tab1" class="selected">{PHP.L.Structure}</a></li>
    <li><a href="{PHP.sys.request_uri}#tab2">{PHP.L.Metadata}</a></li>
    <!-- Вкладки для перевода структуры -->
    {STRUCTURE_UPDATE_I18N_TABS_HEADERS}
</ul>
```

Добавьте тело вкладок:
```html
<div class="tab-box">
    ... (стандартные вкладки tab1, tab2) ...

    <!-- Содержимое вкладок перевода структуры -->
    {STRUCTURE_UPDATE_I18N_TABS_BODY}
</div>
```

## 3. Как это работает на сайте

1. Когда администратор добавляет или редактирует страницу/категорию, он видит вкладки (например, **EN**, **DE**).
2. Заполнив поля на этих вкладках и сохранив форму, переводы запишутся в таблицы БД.
3. Когда посетитель просматривает сайт:
   - Если его язык совпадает с языком по умолчанию (`$cfg['defaultlang']`), выводится базовый текст из основных таблиц.
   - Если его язык переключен (например, на `en`), плагин автоматически заменяет заголовки, описания, тело страниц и метатеги на переведенные значения.
   - Если для какого-то элемента перевод на выбранный язык отсутствует, сработает **Fallback (резервный вариант)** — страница отобразится на языке по умолчанию.

## 4. Переключатель языков на сайте

Для того чтобы гости и посетители сайта могли переключать язык, добавьте тег `{HEADER_I18N_SELECTOR}` в шаблон шапки вашего сайта (`skins/ваша_тема/header.tpl`).

Этот тег выводит блок со ссылками на доступные языки перевода (активный в данный момент язык автоматически скрывается, показываются только альтернативные варианты для перехода).

Например, при просмотре сайта на русском языке переключатель выведет ссылку только на английский:
```html
<div class="i18n-switcher">
    <a href="/some-page?id=123&lang=en" class="i18n-lang-en">EN</a>
</div>
```

Вы можете стилизовать этот виджет в CSS вашей темы, например:
```css
.i18n-switcher a {
    text-decoration: none;
    margin: 0 5px;
    font-weight: bold;
}
```
