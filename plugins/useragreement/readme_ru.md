# Инструкция по установке плагина `useragreement`

Плагин выводит на форму регистрации чекбокс для подтверждения согласия с пользовательским соглашением, политикой конфиденциальности и согласием на обработку персональных данных.

## Шаг 1. Установка плагина
1. Зайдите в **Панель управления** Seditio.
2. Перейдите во вкладку **Инструменты / Плагины**.
3. Найдите плагин **User Agreement** и нажмите **Установить**.

## Шаг 2. Настройка плагина
1. После установки нажмите на кнопку **Настройка** у плагина **User Agreement** (или перейдите в *Конфигурация* -> *User Agreement*).
2. Заполните ссылки на необходимые документы:
   - **Ссылка на Пользовательское соглашение**
   - **Ссылка на Политику конфиденциальности**
   - **Ссылка на Согласие на обработку персональных данных**
   
   *Вы можете указать как все 3 ссылки, так и любую их комбинацию. Текст согласия и ошибки сформируется автоматически.*

## Шаг 3. Интеграция в шаблон
Чтобы чекбокс отображался на форме регистрации, вам необходимо добавить тег `{USERS_REGISTER_AGREEMENT}` в шаблон регистрации вашего скина.

1. Откройте файл шаблона регистрации в вашем активном скине:
   `skins/<имя_вашего_скина>/users.register.tpl`
   *(Если в скине нет этого файла, скопируйте его из `modules/users/tpl/users.register.tpl` в папку вашего скина).*

2. Найдите место перед кнопкой отправки формы (обычно рядом с кнопкой `<input type="submit" ...>`).

3. Добавьте следующий код:
   ```html
   <li class="form-row">
       <div class="form-field-100">
           {USERS_REGISTER_AGREEMENT}
       </div>
   </li>
   ```
4. Сохраните файл шаблона и очистите кэш сайта в панели управления.

## Шаг 4. Стилизация (необязательно)
Для того чтобы ссылки на документы выглядели подчёркнутыми и аккуратными, вы можете добавить следующие CSS-стили в файл стилей вашего скина (например, `index.css` или `custom.css`):

```css
.useragreement-label a {
	text-decoration: underline;
}
.useragreement-label a:hover {
	text-decoration: none;
}
```
